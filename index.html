What do these mean?

OFF: wheels up. ON: wheels down. OUT: pushback/blocks out. IN: blocks in.

Each stamp saves both your local time and UTC time to local storage for offline retrieval. Totals show HH:MM, decimal hours (e.g. 1.75), and tenths (e.g. 1.8).

Install

This is a Progressive Web App. On iPhone: Share → Add to Home Screen. On desktop Chrome/Edge: install from the address bar.
